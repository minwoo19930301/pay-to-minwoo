# 런칭 체크리스트

## 사업/정산

- 사업자등록 여부 확인
- 대표자명과 정산 계좌 명의 일치 여부 확인
- 환불 계좌 및 환불 정책 준비
- 판매 불가 업종 여부 확인

## 화면/정책

- 이용약관
- 개인정보처리방침
- 환불 정책
- 고객 문의 채널
- 영문 정책 페이지

## 결제

- 테스트 키와 라이브 키 분리
- success URL, fail URL 분리
- 웹훅 서명 검증
- 중복 웹훅 방지
- 결제 성공 상태 idempotency 적용

## 운영

- 관리자에서 결제 내역 조회
- 실패 재시도 로그
- 환불 처리 이력
- 차지백 대응 자료 저장
